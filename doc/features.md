# Pavona Features Overview

The Pavona project is an open-source project that provides improvements in utility, flexibility, and standards alignment for secure open silicon.

In alignment with this mission, the project provides:
- High quality hardware modules with security-relevant functionalities and configurability
- Accompanying firmware and software that uphold rigorous security standards
- A tooling flow that enables generation of additional collateral from source files
- Hardware and software testing to maintain repo asset quality

## Basic Pavona Usage

Pavona's contents are meant to be flexible, able to be integrated with external workflows.

## Quality Hardware

Pavona provides an assortment of IP blocks, primitives, and reference top-level designs.

## Quality Software

Accompanying the hardware collateral, Pavona offers device-side and host-side software libraries and tests.
